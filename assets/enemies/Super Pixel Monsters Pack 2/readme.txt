// HOW TO USE THIS ASSET PACK //
////////////////////////////////

Here are a few pointers to help you navigate and make sense of this zip file.

- In the root folder, you will find folders named PNG and spritesheet.

- The PNG folder contains all the monster characters separated into their own folders, with the frames as individual PNG files.

- Some monsters may have multiple animations. In this case, each animation will have its own folder within the monster's folder.

- The PNG/effects folder contains effect animations that could be used with the monsters as death animations.
	- For more effects packs, check out my store page!

- The spritesheet folder contains all the monster characters separated into their own folders, but with the frames of all that character's animations packed into a single image. A metadata file is alongside each spritesheet which may be used to parse the image.

- Recommended animation FPS: 15 (66.7 ms/frame)
